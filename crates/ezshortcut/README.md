# ezshortcut

A cross-platform library for creating desktop/app shortcuts.
See crate documentation for more information.

## Features

- Create shortcuts for common platforms
- Add shortcut to Start Menu/Applications/etc, and attempt to refresh list
- Automatic escaping/sanitization of names/paths

## Platforms

- Windows (`Win32`)
- macOS (`.app`)
- Linux/BSD/Unix (`.desktop`)
